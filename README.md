# task-manager
NodeJS task management server with connection to mongoDB.

> **STACK**
> - MongoDB
> - Mongoose
> - Expresss
> - Sendgrid
> - Multer
> - Supertest

## Instalation
*	**Clone and install packages**
```
git clone https://github.com/zakve/task-manager.git
cd task-manager
yarn install
yarn run dev
```

> Before run add `.env` file to root with
  
```
PORT=****
MONGODB_URL="****"
TOKEN_SECRET_HASH="****"
SENDGRID_API_KEY="****"
```
